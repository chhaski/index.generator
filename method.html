About



A python generator that produces speculative indexes, using NLP to map textual data spatially.

This ongoing project explores the index as a creative infrastructure for mapping knowledge and associative relationships within information. The system generates unique spatial layouts of text on the page, where each index entry's horizontal position is determined by its semantic proximity to others in the dataset (as determined by an approximate nearest neighbors algorithm), forming a multiply-determined visual topology of relationships.

The corpus for this instance of the project is comprised of the indexes of 3 books selected by the artist from the Cybernetics Library’s ‘Uncomputable’ collection:

········Updating to Remain the Same: Habitual New Media···Wendy Hui Kyong Chun
········Code and Clay, Data and Dirt: Five Thousand Years of Urban Media···Shannon Christine Mattern
········ The Moment of Complexity: Emerging Network Culture···Mark C. Taylor

The webpage provides a visual and textual interface to explore a dataset of index entries and their associated metadata. An interactive 3D t-SNE visualization displays the vector space, each point representing an index entry. Users can generate a new speculative index by clicking a button, which highlights the corresponding selected entries in the visualization, constructing a unique shape each time it is regenerated.

Methodology:


········pre-processing
················make each index structure uniform
················parse, reassemble
········processing
················create vectors for each sequence of text (subentry) in the corpus
················reduce dimensionality of each vector with t-SNE 
················save the x-values of t-SNE visualization of vectors in 2-d space as an array, map it to a better range
················create dictionary of each subentry and its corresponding vector
········generating
················randomly chosing from dataset of vector-spaced subentries
················the position of each subentry on the page is determined by the x-position of the vectors in 2-d space as determined by t-SNE
